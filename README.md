
iOS BaasBox SDK
===============

BaasBox iOS Swift 3 SDK.

Documentation: [Docs](Link)

#Installation

Drag the files inside BaasBoxSDK in your project, if asked to create a Bridging header do it.
Add the following line in your Bridging Header:

	#import "BaasBox.h";

And you are setup
